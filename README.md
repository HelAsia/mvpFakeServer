# MVP Fake Server

### Fake server for MVP example based on repository: https://my-json-server.typicode.com/

**BASE_URL:** https://my-json-server.typicode.com/HelAsia/mvpFakeServer/

**ENDPOINTS:**
    - @POST("users/")
    - @PUT("users/{userId}/")
    - @GET("users/")
    - @DELETE("users/{userId}/")
